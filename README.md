# InterpretReflect OS

The Interpreter Operating System for wellbeing, skills, and growth.

Built with:
- Next.js (App Router)
- Tailwind CSS
- Supabase (auth, database, RLS)
- Multi-agent coaching via Catalyst
- Skills OS: goals, assessments, practice queue
- Wellbeing OS: emotional load, burnout drift, support stack

## Local Dev
npm install
npm run dev

## Deployment
Auto-deploys via Vercel.

## Env Vars
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
